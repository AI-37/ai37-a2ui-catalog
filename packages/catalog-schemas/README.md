# @ai37/a2ui-catalog-schemas

Каноничные **Zod-схемы** и метаданные A2UI-каталога AI37, плюс экспорт **JSON Schema**.
Источник истины по составу каталога компонентов (включая интерактивные `ChoiceCard` / `FormCard`).

```bash
npm i @ai37/a2ui-catalog-schemas
```

```ts
import { CATALOG_ID, componentDefinitions } from "@ai37/a2ui-catalog-schemas";
```

Рендереры для React живут в [`@ai37/a2ui-catalog-react`](https://www.npmjs.com/package/@ai37/a2ui-catalog-react),
Pydantic-модели — в Python-пакете каталога.

Лицензия: Apache-2.0.
